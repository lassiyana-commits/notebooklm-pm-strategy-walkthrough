# 📸 Screenshots

This folder is where the real screenshots from a live NotebookLM walkthrough go. The main [README](../README.md) already links to these exact filenames — just add your images with matching names and they'll appear automatically on GitHub.

Suggested screenshots to capture, one per step:

| Filename | What to capture |
|---|---|
| `step-1-add-sources.png` | The NotebookLM "Add Source" screen with your 3 company sources listed |
| `step-2-save-as-note.png` | An AI answer in NotebookLM's chat with the "Save as Note" button visible |
| `step-3-generate-report.png` | The source list with only the 3 Notes selected, and the generated Strategic Comparison Report |
| `step-4-gemini-prompt.png` | Gemini with the Report pasted in and the "Lead Product Manager" prompt submitted |
| `step-5-day-in-life-story.png` | Gemini's day-in-the-life response |
| `step-6-video-overview.png` | NotebookLM's video/audio overview generation screen |

**How to add them:**
1. Take the screenshot (crop out anything sensitive, like personal emails or account names).
2. Save it with the exact filename from the table above.
3. Drop it into this folder, replacing the placeholder.
4. Commit and push — GitHub will render it inline in the README automatically.
